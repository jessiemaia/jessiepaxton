Summit

Application where user selects tasks from the task manager, saves them, and checks them off upon completion. Upon task completion, it's point value is added to user's total score. On the home screen, the user can watch their progress bar climb up a mountain. With team capability, users can add other users to a team and see teammates point value. Vue.js client side - server side includes Node.js - MongoDB setup with Mongoose ORM.
